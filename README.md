# STM32-Smart-Alert-and-Indication-System
A Smart Alert and Indication System built using the STM32F401RE microcontroller. The project demonstrates GPIO input/output control by interfacing push buttons, LED, buzzer, relay, and a common cathode 7-segment display using STM32CubeMX, STM32CubeIDE, Embedded C (HAL), and Proteus simulation.
